# LCPS NMSF Snapshots

Count-only manual snapshots from official Loudoun County Public Schools National
Merit Semifinalist releases. Student names are intentionally omitted.

Class 2023 and 2024 school-attributed snapshots come from archived LCPS
Blackboard pages. Class 2026 is also school-attributed. The Class 2025 official
release is retained as a total-only, source-incomplete snapshot for
reconciliation; it does not create school-level observations or verified zeros.
