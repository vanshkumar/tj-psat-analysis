# FCPS NMSF Source Snapshots

This directory archives source-derived snapshots for FCPS National Merit
Semifinalist releases used by the NMSF observation layer.

The official FCPS pages list individual student names. These snapshots preserve
the school-level count evidence needed for this project while intentionally
omitting student names. Each CSV keeps the source URL, title, publication date,
retrieval date, and count rows transcribed from the official page.

The source manifest records the SHA-256 hash of each snapshot file.
